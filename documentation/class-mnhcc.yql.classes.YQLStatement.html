# bindParam( mixed $parameter, mixed & $variable, integer $data_type = null )

Binds a value to a parameter Binds a value to a corresponding named or question mark placeholder in the YQL statement that was used to prepare the statement.

Binds a value to a parameter Binds a value to a corresponding named or question mark placeholder in the YQL statement that was used to prepare the statement.

Parameters

$parameter
mixed
$parameter int|string Parameter identifier.

For a prepared statement using named placeholders, this will be a parameter name of the form :name. For a prepared statement using question mark (?) placeholders, this will be the 1-indexed position of the parameter.
Masking question mark with \?
Example:
<code><?php $qery = 'select description from rss(5) where url = "http://example.com/rss\\?parameter=value'; ?></code>
